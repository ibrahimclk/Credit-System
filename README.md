# Paycore Java Spring Bootcamp Project
Credit Application System with Spring 

##Used Technologies

Java 8

Spring Boot

JPA

Swagger

MapStruct

Lombok

PostgreSQL

Mockito

Maven

##Run

Port: 8080

git clone https://github.com/nejlasahin/credit-application-system.git

$ cd Credit-System

$ mvn clean install

$ mvn spring-boot:run

##Swagger Ui

http://localhost:8080/swagger-ui.html
